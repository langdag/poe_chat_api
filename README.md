## Prerequisites

- **Go**: Ensure you have [Go](https://golang.org/doc/install) installed (version >=1.18).
- **PostgreSQL**: Install [PostgreSQL](https://www.postgresql.org/download/), and ensure it's running.
- **Git**: Install [Git](https://git-scm.com/).
